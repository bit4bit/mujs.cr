# mujs

Crystal Lang lightweight Javascript interpreter using MuJS.

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     mujs:
       github: bit4bit/mujs.cr
   ```

2. Run `shards install`
